# 行为准则

## 我们的承诺

为了营造一个开放、包容的社区，我们承诺让所有参与者都能享受无骚扰的体验，无论年龄、体型、残疾、种族、性别认同、性取向、国籍、个人外貌、宗教或性取向如何。

## 我们的标准

有助于创建积极环境的行为包括：

- 使用友好和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表现出同理心

## 违反行为

禁止的行为包括：

- 使用性暗示的语言或图像
- 骚扰、侮辱或歧视性评论
- 公开或私下的骚扰行为
- 未经许可发布他人的个人信息
- 其他可能被认为不适当的行为

## 执行

项目维护者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、维基编辑、问题和其他贡献。

## 范围

本行为准则适用于项目空间以及在公共场合代表项目或其社区的个人行为。

## 报告

如果您遇到违反行为准则的情况，请联系项目维护者。
